# CrossChat — Features Overview

## Current Features (v1.7.0)

- Hub-based messaging through rooms
- WebSocket connections for Claude Code instances
- Task delegation and lifecycle management (delegate → claim → accept → update → complete)
- Room-based messaging with @mentions
- Peer discovery and status management
- Session management and lifecycle hooks
- Request digest for room history
- Dashboard for monitoring
