# Backlog

Items parked here are not scheduled for any sprint. They may be pulled into a sprint during `/discuss` planning sessions.

## When to park items here
- Feature ideas that are valid but not prioritized
- Tasks deferred from a sprint due to scope reduction
- Nice-to-haves discovered during implementation
- External dependencies that are blocked

## When to pull items
- Sprint planning identifies capacity for additional work
- A blocking dependency is resolved
- Priority shifts make the item urgent
